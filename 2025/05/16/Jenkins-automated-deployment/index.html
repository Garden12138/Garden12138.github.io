Jenkins自动化部署

docker方式安装Jenkins

  • 拉取Docker仓库的Jenkins镜像

  • 运行Jenkins容器

  • 浏览器输入Jenkins应用地址（IP为示例服务器IP）

  • 获取管理员初始化密码，输入登录页面中

  • 安装推荐插件（需翻墙安装）

  • 创建管理员账户

  • 配置全局安全配置，取消勾选跨站请求伪造保护

Docker发布SpringBoot应用

  • 插件管理

    • 安装mvn插件

  • 全局工具配置

    • Maven配置，优先在宿主机对应Jenkins容器挂载的文件目录下，安装Maven

    • JDK配置，选择自动安装方式

    • Git配置，选择自动安装方式

  • 系统管理

    • 设置maven全局属性

    • 设置SSH服务器配置

  • 新建任务

    • 创建流水线

    • 配置流水线

  • SpringBoot项目/build/pro/目录下编写JenkinsFile

  • SpringBoot项目/build/pro/目录下编写DockerFile

  • SpringBoot项目/build/pro/目录下编写run.sh

  • 发布SpringBoot应用

Nginx发布前端应用

  • jenkins安装nodejs插件

  • jenkins配置nodejs全局工具

  • jenkins系统配置SSH Server

  • 编写jenkins任务

    • 创建自由风格软件项目

    • General设置

    • 源码管理设置

    • 构建环境设置

    • 构建设置

    • 构建后操作设置

  • 编写nginx转发配置

    • 编写nginx.conf配置文件

    • 刷新nginx.conf配置文件

    • 重启nginx

  • 浏览器打开前端地址，如http://39.108.168.201:3000/
